# PHP Socket Server

A brief description of what this project does and who it's for


## Run Locally

Clone the project

```bash
  git clone https://github.com/arlo-peace/php-socket
```

Go to the project directory

```bash
  cd php-socket
```

Install composer

```bash
  composer install
```

Run Server
```bash
  php bin/server.php
```
